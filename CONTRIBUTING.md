# Contributing to Quantum Mechanics Gallery

Thank you for your interest in contributing!

## How to Contribute

1. **Fork** this repository
2. **Create a branch**: `git checkout -b topic/your-topic-name`
3. **Add your content** following the template below
4. **Submit a Pull Request** with a clear description

## Topic Template

Every topic file in `topics/` should follow this structure:

```markdown
# 🔷 Topic Title

## Fact Sheet

| Property | Details |
|----------|---------|
| Key1 | Value1 |

---

## Core Concept

[Explanation with equations in code blocks]

## Key Equations

[Equations]

---

## 📖 References
- Author (Year). Title. Journal.
- [arXiv link if applicable]
```

## Guidelines

- Use LaTeX-style notation in code blocks for equations
- Always cite original papers
- Keep fact sheets concise (< 8 rows)
- Add arXiv links where available
- Figures should be placed in `figures/` directory
