## 📚 What is this?

This repository is a structured reference gallery for **Quantum Mechanics**, collecting:
- Core postulates and axioms
- Key equations and their derivations (in LaTeX)
- Architecture diagrams of quantum systems
- Fact sheets for major topics
- Links to original papers and resources

Think of it as a **visual and mathematical cheat sheet** for quantum mechanics — from Schrödinger's equation to quantum field theory.
