# 🎯 Heisenberg Uncertainty Principle

## Fact Sheet

| Property | Details |
|----------|---------|
| Formulated by | Werner Heisenberg (1927) |
| Type | Fundamental quantum limit |
| NOT due to | Measurement disturbance (Robertson-Schrödinger form is more general) |
| Generalization | Robertson uncertainty relation |

---

## Position-Momentum Uncertainty

```
Δx · Δp ≥ ℏ/2
```

- Δx = standard deviation of position
- Δp = standard deviation of momentum
- Minimum uncertainty achieved by **Gaussian wave packets**

## Energy-Time Uncertainty

```
ΔE · Δt ≥ ℏ/2
```

Note: time is a parameter in QM, not an operator — this relation has a different interpretation.
